# stm32h7xx
apollo stm32h7xx development board

## resources on board
### MCU
  stm32h743, 400mhz, 1Mbyte SRAM, 2MByte Flash.
### SDRAM
  W9825G6KH, 32MByte, 133mhz.
### QSPI Flash
  W25Q256, 32MByte, 104mhz.
### NAND Flash
  512Mbyte.
### LCD
  NT35510,LCD MPU initerface, 565 color, 800x480.
### SD Card
  SDIO interface.
